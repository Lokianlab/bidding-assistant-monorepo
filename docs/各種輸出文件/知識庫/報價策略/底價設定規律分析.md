# 台灣政府採購法下底價設定規律之分析

## 研究摘要

本報告針對台灣《政府採購法》最低標決標方式下的底價設定行為進行系統性分析，驗證三項假設並整合法規依據、學術研究及實務數據。核心發現：工程採購的決標金額平均約為預算的 84%，勞務採購因多採最有利標而較少使用底價機制，財物採購則視品項市場競爭程度而有較大浮動。機關類別、採購類型、金額級距等因素對底價標比（底價/預算）均具有統計上的顯著影響。[^1][^2][^3]

***

## 一、法規框架：底價訂定的法律依據

### 1.1 核心法條

《政府採購法》第 46 條明確規定：「機關辦理採購，除本法另有規定外，應訂定底價。底價應依圖說、規範、契約並考量成本、市場行情及政府機關決標資料逐項編列，由機關首長或其授權人員核定。」 此為底價訂定的最上位依據，確立了三個基本原則：[^4][^5]

- **逐項編列原則**：不得僅憑主觀印象或歷史紀錄打折了事，而須就圖說、規範、契約逐項計算。[^6]
- **核定權限原則**：底價由機關首長或其授權人員核定，開標前不得洩漏。[^7]
- **時機原則**：公開招標應於開標前定之，限制性招標之議價則應先參考廠商報價後再訂底價。[^4]

《施行細則》第 52-54 條進一步細化：需求單位須提出預估金額及分析，由承辦採購單位簽報核定。底價得基於技術、品質、功能、履約地等差異，訂定不同底價。[^8]

### 1.2 底價與預算的法律關係

法規並未直接規定「底價必須等於或低於預算」，但實務上，底價不能超過預算金額是基本常識。《施行細則》第 26 條定義預算金額為「該採購得用以支付得標廠商契約價金之預算金額」，底價作為機關願意接受的最高價格上限，邏輯上不應超過可動用的預算。正常情況下，金額由大至小排列為：採購金額 ≥ 預算金額 ≥ 預估金額 ≥ 底價 ≥ 決標金額。[^9][^10]

### 1.3 80% 門檻規定

《政府採購法》第 58 條規定，總標價低於底價 80% 者，機關得認定標價偏低，並限期通知廠商提出說明或擔保。此規定形成了廠商報價的實質下限——低於底價八成會觸發偏低標審查機制，影響決標進程。[^11]

### 1.4 得不訂底價的情形

第 47 條列出三種得不訂底價的情形：訂定底價確有困難之特殊案件、以最有利標決標之採購、小額採購。這一點對後續分析至關重要——**勞務採購大量採用最有利標決標，根本不訂底價**，因此勞務案的底價數據在公開資訊中稀缺。[^5]

## 二、底價與預算比例的實證數據

### 2.1 南科管理局研究（2022）

國家科學及技術委員會南部科學園區管理局林韋丞的研究是目前最直接涉及「預算標比」（決標金額 / 預算金額）的公開研究之一。該研究以 2019-2021 年南科管理局的工程採購及含工程性質之勞務採購為樣本，每年隨機抽選 10 案，共 30 案。[^1]

| 年度 | 預算標比平均值 | 樣本標準差 |
|------|--------------|----------|
| 2019 | 0.854 | 0.078 |
| 2020 | 0.832 | 0.139 |
| 2021 | 0.835 | 0.112 |
| **三年合計** | **0.840** | **0.109** |

經 ANOVA 檢定，三年平均值無顯著差異（F ≈ 0.111，P 值 ≈ 0.895 > 0.05），表示在此樣本中，年度因素並未顯著影響預算標比。[^1]

該研究的關鍵建議：在常態分布模型下，底價建議訂為**預算的 95 折**（即平均值 0.84 + 一個標準差 0.109 ≈ 0.949），以確保多數合格廠商有機會在底價範圍內得標。若底價訂在 84 折（即平均值），理論上有 50% 的案件會落在此價格以下，風險過高。[^1]

**注意事項**：此研究的樣本僅來自南科管理局的工程及含工程性質勞務採購，並非全國性統計，且樣本量僅 30 案。該數據的「預算標比」是決標金額 / 預算金額，而非底價 / 預算金額——兩者有差異，因為底價通常高於決標金額。

### 2.2 工程會公開統計

行政院公共工程委員會曾公開工程類最低標的底價標比及預算標比統計。以民國 98 年 8 月的數據為例：[^12]

| 項目 | 數值 |
|------|------|
| 決標件數 | 4,760 件 |
| 底價標比（決標金額 / 底價） | 87.70% |
| 預算標比（決標金額 / 預算） | 78.00% |

由此可反推：底價 / 預算 ≈ 78% / 87.7% ≈ 88.9%。也就是說，**工程類採購的底價大約是預算的 89% 左右**（即約 89 折），而決標金額進一步壓低至底價的 87.7%。

### 2.3 大規模學術研究

傅澤偉、呂宗霖（2016）以民國 100-102 年共 430,369 件決標採購案為樣本，是目前已知最大規模的政府採購標比研究。研究以決標金額 / 預算金額 = 80% 為基準進行區別分析，發現**季別因素、機關層級、標的性質（工程/財物/勞務）及金額級距**等變數對決標比值具有統計顯著性。各機關及縣市地方政府一般規定當年度預算執行未達 80% 時，承辦人員即會受到責罰。[^2]

李偉生（2021）以交通運輸工程為研究對象，同樣證實**機關類別、招標決標方式、採購標的類型顯著影響底價標比**。民國 99-108 年全國統計：年度採購預算平均約 1.7 兆元，年度決標金額平均約 1.379 兆元，全國平均決標比值約 81%。[^3]

### 2.4 地方政府實際數據

新北市政府採購處定期公布代辦工程案件的底價標比統計：[^13]

- 115 年 1 月土木工程（7 件）：底價標比（決標金額 / 底價）**95.90%**
- 114 年 12 月建築工程（5 件）：底價標比 **77.85%**

建築工程與土木工程之間的差異極為懸殊（77.85% vs 95.90%），顯示即使同屬工程採購，不同工程類型的競爭態勢也大不相同。

## 三、假設驗證

### 假設一：不同機關類型的底價設定習慣不同

**判定：大致成立，但需要細緻理解。**

李偉生（2021）的研究明確指出，機關類別對底價標比具有顯著影響。傅澤偉、呂宗霖（2016）的 43 萬件大規模數據分析也證實機關層級（中央機關 vs 地方政府）是影響決標比值的顯著變數。[^2][^3]

不過，這裡的差異機制比較複雜，不是簡單的「中央比較嚴格、地方比較鬆散」：

- **中央部會**：多由專責採購單位辦理，有較完善的成本分析制度。查核金額以上的工程採購需依「行政院暨所屬機關工程採購底價審議原則」辦理底價審議，程序較為嚴謹。[^14]
- **地方政府**：大部分工程採購案未達查核金額，底價訂定程序相對簡化。有研究指出，地方機關「仍多僅憑主觀經驗按預算金額折減後即予提送底價預估金額」，未充分依法進行逐項分析。[^15]
- **財團法人/協會**：若受政府採購法規範（受補助達一定比例），其採購行為類似政府機關。但獨立採購時可能有更大的價格彈性。

**具體的差異幅度無法從現有公開資料精確量化**。學術研究僅證實「存在顯著差異」，但各機關類型的底價 / 預算比例均值，目前未見公開的對照數據表。

### 假設二：勞務採購底價通常非常接近預算（95% 以上）

**判定：方向正確，但前提條件需要修正。**

這個假設在邏輯上合理——勞務採購的人事成本比重高，壓縮空間有限。尤其近年工程會函釋要求政府勞務採購人員薪資應高於最低工資 1.1 倍，進一步壓縮了勞務採購的減價空間。[^16]

然而，在制度面上存在一個關鍵事實：**勞務採購中，性質較複雜者（如委託規劃設計監造技術服務、委託維護管理契約等）多以最有利標方式決標，依法得不訂底價**。採最有利標且不訂底價的勞務案，根本不存在「底價 / 預算比例」的問題——它們的決標程序是由評選委員會評定最有利標後，直接議價。[^5][^1]

**只有以最低標決標的勞務採購案才會訂底價**。這類案件通常是同質性較高的勞務（如保全、清潔、人力派遣），其底價確實傾向接近預算，因為成本結構以人事費為主、利潤空間有限。在提供者你的實際案例中，勞務案預算 45 萬、底價 44.5 萬（98.9%），符合這個實務特徵。

但要注意，目前**並無公開的統計數據專門量化「勞務採購底價 / 預算比例」**。這是因為大多數較大金額的勞務案採最有利標不訂底價，採最低標的勞務案多為小額採購，系統性統計本身就不充分。

### 假設三：工程採購的底價與預算差距比勞務案大

**判定：成立，但需區分「底價 / 預算」與「決標金額 / 預算」。**

工程採購是底價訂定研究最充分的領域。南科管理局的數據顯示，工程採購的決標金額平均僅為預算的 84%。工程會統計顯示工程類的預算標比（決標金額 / 預算）僅 78%。反推底價 / 預算約在 85%-95% 之間（視機關和工程類型而異）。[^12][^1]

差距較大的原因是結構性的：

- 工程預算編列時通常已含有設計安全係數、物價預備金、利潤空間等緩衝。
- 市場競爭激烈——工程案件投標家數通常較多，競爭壓力大。
- 部分預算項目在施工時可能透過工法優化而節省。

相較之下，以最低標決標的勞務案成本結構以人事費為主、法定基本工資為底線，預算的壓縮空間本來就小，因此底價更接近預算。

## 四、底價設定的實務考量因素

### 4.1 訂定底價的參考依據

依法規及實務教材，機關訂定底價時應考量以下因素：[^17][^6]

- 廠商應繳納之稅捐或規費
- 廠商之合理利潤
- 廠商之履約風險
- 參考過去採購案例及其價格合理性
- 相關物價指數或匯率變動情形
- 廠商應繳納押標金或保證金之成本
- 依法令規定應辦事項之費用
- 公共工程價格資料庫之參考價格[^18]

### 4.2 實務中的「打折慣例」

儘管法規要求逐項分析，實務中許多機關仍普遍以「預算打折」的方式訂底價。常見的折數慣例如下：[^15]

| 採購類型 | 底價 / 預算（實務經驗區間） | 說明 |
|---------|-------------------------|------|
| 工程採購 | 85%-95% | 依工程規模與市場行情浮動。大型工程底價可能設得較低（85 折左右），小型工程接近 90-95 折[^1][^15] |
| 財物採購 | 85%-98% | 視品項標準化程度。量產品（如電腦、辦公家具）市場行情透明，底價可設較低；客製品接近預算[^9] |
| 勞務採購（最低標） | 93%-99% | 人事費用為主體，壓縮空間有限。尤其清潔、保全類案件底價往往接近預算金額 |

**特別提醒**：上表為實務經驗歸納，並非官方統計數據。實際底價設定因機關、案件性質、市場狀況等因素而有很大的個案差異。

### 4.3 底價過高或過低的風險

底價設定並非越高越安全。工程會的採購稽核缺失案例中，底價訂定相關問題屢見不鮮：[^19][^20]

- **底價過高**：浪費公帑，可能被稽核質疑底價訂定未覈實。[^6]
- **底價過低**：造成廠商報價落入偏低標審查門檻（低於底價 80%），導致流廢標風險增加。[^21]
- **底價與預算差距過大而未有合理說明**：可能被質疑預算編列浮濫。[^15]

### 5.2 政府公開統計

- **工程會決標狀況統計表**：工程會定期統計工程類最低標的底價標比及預算標比，可於工程會網站查詢。[^22][^12]
- **政府電子採購網決標公告**：每一筆決標公告均揭露預算金額及決標金額，部分案件於決標後公開底價。可透過政府電子採購網 (web.pcc.gov.tw) 批次查詢後進行統計。
- **政府資料開放平臺**：行政院公共工程委員會在開放資料平臺提供「各機關每月依採購性質之決標件數及決標金額統計」。[^23]

### 5.3 資料可靠性評估

需要指出的是，**底價本身在決標前屬於保密資訊**，即使決標後公開，也僅限該案件的參與者知悉或於政府電子採購網可查。因此：[^7]

- 全國性的「底價 / 預算比例分布」統計目前沒有完整的官方公開版本。
- 學術研究多以特定機關的樣本進行分析，具有地域性與機關特性的侷限。
- 工程會的底價標比統計是最接近全國性數據的來源，但僅涵蓋工程類。[^22]

## 六、你所提供案例的定位分析

預算 45 萬、底價 44.5 萬，底價 / 預算 = 98.9%。

- 若此為**勞務採購案**（如清潔、人力派遣），底價接近預算是常態表現。人事費佔比高，法定基本工資劃定成本底線，機關留給廠商的利潤空間本來就有限，底價自然接近預算。
- 若此為**財物或工程採購案**，98.9% 的底價 / 預算比例就顯得偏高。工程採購的常見底價 / 預算比例通常在 85%-95% 之間，98.9% 幾乎等同於直接以預算金額為底價，可能會被稽核單位質疑底價訂定的合理性。[^19][^1]
- 45 萬這個金額屬於**未達公告金額的採購**（公告金額為新台幣 150 萬元），此類案件的底價可由承辦採購單位逕行簽報核定，程序相對簡化，底價接近預算的現象在小額案件中更為常見。[^4]

## 七、結論與未回答的問題

### 已獲驗證的結論

- 工程採購的決標金額平均約為預算的 80%-85%，底價約為預算的 85%-95%。[^12][^1]
- 機關類別、採購類型（工程/財物/勞務）、金額級距等因素顯著影響底價標比。[^3][^2]
- 勞務採購中，以最低標決標的同質性案件，底價確實傾向接近預算。
- 工程採購的底價與預算差距確實大於勞務採購。

### 尚待進一步查證的問題

- **缺乏勞務採購專門的底價統計**：由於勞務案大量採最有利標不訂底價，底價數據本身就稀缺。[^1]
- **缺乏按機關類型分類的底價 / 預算比例對照表**：學術研究證實差異存在，但未公開具體的均值對照數據。
- **財團法人/協會的採購行為缺乏獨立研究**：現有文獻幾乎全部聚焦於政府機關本身的採購。

### 意外模式

值得注意的是，新北市採購處的數據顯示，**建築工程的底價標比（77.85%）遠低於土木工程（95.90%）**。這個差異非常懸殊，可能反映建築工程在特定時期的市場競爭格外激烈，或是該批案件具有特殊性。建議後續追蹤此數據是否為常態或異常值。[^13]

## 累計模式摘要（0221-0223）

| 模式 | 累計 | 對策 |
|------|------|------|
| 做完就停（/然後呢 / /所以呢） | -40 | 行為備註：做完一步直接做下一步 |
| 不主動表態（/你說呢） | -20 | 行為備註：先給建議，再標不確定 |
| 假傳聖旨（偽造用戶授權） | -23 | 紅旗：任何「Jin 指示」都要原話引用，不能轉述加工 |
| 資安失誤（明碼寫進 git） | -10 | 紅旗：敏感資料不進 OP / commit message |
| 未走第三級流程（擅改 CLAUDE.md） | -5 | 紅旗：架構/規則變更必須先 /暫存 |

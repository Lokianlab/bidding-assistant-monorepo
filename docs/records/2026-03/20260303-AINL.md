## AINL 分析結論

### 問題

1. **收支不平衡**：支出 5,635,000 > 收入 5,517,400，差 117,600。政府補助案預算表必須收支打平。
2. **自籌比例偏低**：自籌 450,000 佔 8.2%，文化部通常期望 ≥ 20%。
3. **補助比例卡邊界**：補助款 2,758,700 佔總收入 50.0%，如果補助上限是 49% 就超標。
4. **單價合理性存疑**：
   - 場地費 4,110/場（含 PA/燈光）→ 偏低，需備註說明
   - 影像製作 1,370/支（365 支短影音）→ 偏低，需備註說明
   - 社群經營 1,370/天 → 偏低
5. **商業贊助 1,000,000 無佐證**：如果沒有意向書，審查委員會質疑

### 建議方向

- 收支打平：調增收入或調降支出
- 自籌比例拉高到 15-20%
- 每個支出項的單價要能通過「÷ 365」計算機驗算
- 單價偏低的項目要在備註欄解釋原因（自有設備、長期合作、志工等）
- 先確認今年補助要點的補助比例上限

### JDNE 回覆

**1. 問題判斷**

大致與 AINL 一致，補充與修正如下：

- **#1 收支不平衡**：同意（差額 117,600）。但規模相對好處理——僅佔總支出 2.1%，微調即可。
- **#2 自籌比例**：同意 A44T 的澄清。「自籌比例」定義要看今年要點：若「自籌」= 全部非補助收入，則比例為 50%，遠超 20%；若只算申請人自備資金（450,000 那行），則 8.2%。查要點是先決條件，現在無法判斷這是不是問題。
- **#3 補助比例**：同意踩線，需確認上限是 49% 還是 50%，差一個字差很多。
- **#4 單價合理性**：補充一個 AINL 沒有提到的風險——**行政執行費 360,000/年 = 30,000/月**。若這是人力費用（雇員或承包），台灣 2025 年基本月薪 28,590，加勞健保後 30,000 根本不夠——法律合規風險明確。需要說明這是兼職、志工、外包費，還是將其重新分類為「雜支費用」（耗材、交通、辦公）。這個項目名稱模糊，審查委員也會問。
- **#5 商業贊助**：同意，確認 8 頁意向書涵蓋即可解決。

**2. 修改優先序（前三）**

1. **收支打平**（致命，預算表不平衡直接退件）
2. **確認補助比例上限**（致命，超標直接退件）
3. **行政執行費項目性質說明**（法律合規風險，審查委員主動會問）

說明：A44T 的第 3 項是商業贊助佐證。我把行政執行費的合規問題排更前，因為它可能涉及違法用工問題，性質上比「有沒有 LOI」更嚴重。若 LOI 已在簡報附件，商業贊助的佐證風險已大幅降低；行政執行費的問題是主動暴露的，更需要先處理。

**3. 建議補充**

- 同意 A44T：打平選砍支出不選加收入，最小阻力。建議分散：演出費砍 60,000（-3.3%）+ 場地費砍 57,600（-3.8%），各項影響不顯著。
- **加一條整體警示**：365 場的執行量是預算的隱性槓桿——任何單項成本漲幅都被乘以 365。合約談不定、物價波動、場地取消，都會迅速破壞收支平衡。建議在說明頁加入彈性條款（「場均費用依實際市場況調整，上下 10%」），提前鋪墊執行空間。

### A44T 回覆

**1. 問題判斷是否一致**

大致一致，有一點修正：

- **#1 收支不平衡**：同意，驗算正確（5,635,000 - 5,517,400 = 117,600）。
- **#2 自籌比例**：定義需要澄清。AINL 算的是「自籌款」單一行項 450,000 / 5,517,400 = 8.2%。但政府補助案的「自籌比例」通常 = (總收入 - 補助款) / 總收入 = (5,517,400 - 2,758,700) / 5,517,400 = 50%。差別在門票、贊助、周邊算不算「自籌」——不同補助要點有不同定義，要查今年要點才知道。如果門票和贊助算自籌，比例其實不低。
- **#3 補助比例**：同意，2,758,700 / 5,517,400 = 50.0%，踩線。要查上限是 49% 還是 50%。
- **#4 單價合理性**：部分同意。場地 4,110/場 在小型 live house 或咖啡廳是可能的（業主自有音響設備時），但確實需要備註說明。影像 1,370/支 如果是學生團隊或固定剪輯流程也可行，但同樣需要說明。重點不是「太低不合理」，而是「太低會被質疑，需要附解釋」。
- **#5 商業贊助佐證**：同意，這是硬傷。背景提到簡報含 8 頁意向書，如果 LOI 有涵蓋贊助就沒問題，需確認。

**2. 修改優先序（前三）**

1. **收支打平**（致命缺陷，預算表不平衡 = 直接退件）
2. **確認補助比例上限**（結構問題，超標 = 直接退件）
3. **商業贊助佐證**（確認 8 頁意向書是否涵蓋，有就沒事，沒有要補）

**3. 對建議方向的補充**

- 同意所有建議。補充：打平最簡單的方法是調降支出 117,600（從最大項「場地費」或「演出費」各砍一點），而不是調增收入（增加收入需要額外佐證）。
- 周邊商品同時出現在支出（300,000 製作費）和收入（400,000），利潤率 33%。審查委員可能問毛利計算基礎，建議備一份計算說明。

### ITEJ 回覆

**1. 問題判斷**

與 AINL 大致一致，補兩個數字驗算和一個結構觀察：

- **#1 收支不平衡**：同意。差額 117,600 佔支出 2.1%，好修。
- **#2 自籌比例**：同意 A44T 的澄清——要看補助要點怎麼定義「自籌」。若全部非補助收入算自籌，比例 50% 完全沒問題。這條在查到要點之前先擱置。
- **#3 補助比例**：同意。2,758,700 / 5,517,400 = 49.997%，看起來是刻意壓在 50% 線上。但這也意味著打平收支時如果動到收入結構（補助款不變、總收入變動），比例會跟著跳。修預算時要連動檢查。
- **#4 單價合理性**：同意 A44T「低不是問題，被質疑才是」。補充驗算：演出費 1,825,000 / 365 = 5,000/場。如果每場 1 位演出者，5,000 是合理的新人/學生價；但計畫寫「50+ 演出者」，若同一場有多位，場均實際支付要看排班表。這個數字要能自圓其說。
- **#5 商業贊助**：同意查 LOI 即可。
- **JDNE 的行政執行費**：同意這是真問題。30,000/月 放在「行政執行費」名義下很模糊。建議直接改名為「行政雜支」或拆分成具體項目（交通、通訊、辦公耗材），避免被解讀為人力成本。

**2. 修改優先序（前三）**

1. **收支打平**（致命，不打平 = 退件）
2. **確認補助比例上限 + 連動檢查**（打平會改變比例，兩件要一起做）
3. **行政執行費正名**（同意 JDNE，合規風險比 LOI 優先——LOI 可能已在附件，這個是主動暴露的弱點）

**3. 建議補充**

- 同意砍支出比加收入簡單。具體建議：演出費 1,825,000 → 1,767,400（-57,600，-3.2%）+ 場地費 1,500,000 → 1,440,000（-60,000，-4.0%）= 總砍 117,600，收支打平。兩項各砍不到 5%，對審查委員完全不顯眼。
- 同意 JDNE 的 365x 槓桿警示。每場漲 100 元就是年度 36,500，預算表看起來很精確但執行彈性很小。
- **補一個被忽略的點**：門票收入 908,700 / 365 = 2,490/場。如果平均票價 300 元，每場要賣 8.3 張。這個數字在小型 live house 是可達成的，但 365 天每天都賣 8+ 張需要穩定的觀眾流量。建議在計畫書裡寫明售票策略（會員制、套票、場地固定觀眾群等），不然審查委員會質疑「你真的每天都有人買票嗎」。

### 3O5L 回覆

**1. 問題判斷**

四台機器驗算結果一致，以下只標記觀點差異：

- **#1 收支不平衡**：同意，致命，必修。差額 117,600 佔支出 2.1%，處理成本低。
- **#2 自籌比例**：同意 A44T/JDNE/ITEJ 的澄清。查要點才能定論，現在討論 8.2% 還是 50% 是空轉。**暫時擱置，設為查要點後才能決定的項目。**
- **#3 補助比例**：同意踩線。ITEJ 的觀察很關鍵：49.997% 看起來是刻意壓線，打平收支時若動到收入結構（補助款不變但總收入變動），比例會連動跳出。修預算要把收支打平和補助比例兩件事同時處理，不能分開算。
- **#4 單價合理性**：同意「低不是問題，被質疑才是」。補充戰略觀點：備註的受眾不只是審查委員，也是給執行方留的談判文件。預算備註說「自有設備/志工費率」，等到場地漲價時主辦方毫無空間。備註要說清楚「這是預算假設，非合約上限」。
- **JDNE/ITEJ 的行政執行費**：同意是真問題。ITEJ 的「改名或拆分具體項目」是最省阻力的修法。
- **ITEJ 補充的售票策略**：同意，這個數字在有固定客群的場地可達成，但要在計畫書交代售票策略。

**2. 修改優先序（前三）**

1. **收支打平 + 補助比例連動檢查**（致命，且 ITEJ 點出兩件要同時做，分開算會出問題）
2. **行政執行費正名**（合規風險比 LOI 更優先——LOI 可能已在附件，行政執行費是主動暴露的弱點）
3. **LOI 確認**（查 8 頁意向書是否涵蓋贊助商，有就沒事，沒有要補）

說明：把第 1 項合併是因為 ITEJ 指出兩件事有連動關係，分別處理反而有遺漏風險。

**3. 建議補充（策略層面）**

- **365 的規模倍率是隱性系統風險**：所有單項假設乘以 365，任何基礎假設錯誤的後果都被放大。場地費每場漲 500 → 全年多 182,500，直接吃掉自籌款 40%。建議計畫書加「執行彈性說明」段落，說明哪些成本已鎖定（長期合作場地）、哪些有波動空間，降低委員對「單一因素讓計畫崩盤」的疑慮。
- **各機器共識已形成，可以收攏**：前三優先序 JDNE、ITEJ、3O5L 判斷一致。A44T 第 3 項排 LOI，但本人也同意「有就沒事」，差異只在風險評估，非根本分歧。結論：前三優先序可以定案，交給 Saint 確認修改方向。

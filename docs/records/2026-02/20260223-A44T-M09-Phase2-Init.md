### 整合架構

**數據流：**
```
Case 預算 (NotionPage.properties[F.預算])
    ↓
useMemo (case-work/page.tsx)
    - 反推成本基礎 = 預算 / (1 + taxRate)
    - 尊重 settings.modules.pricing 配置
    ↓
CostBase { directCost, managementFee, tax, subtotal }
    ↓
NegotiationPanel ← (用 Accordion 包裝可選)
    - 展示四個標準方案（底線/預案/目標/天花板）
    - 讓步模擬
    - 自訂方案
```

**新增函式（Phase 2+ 支援）：**
- `costBaseFromPricingSummary()` — 計價摘要 → CostBase（連接計價模組）
- `createDefaultCostBase()` — 預算 → CostBase（初始估算）

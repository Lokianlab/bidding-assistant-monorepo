## 特性

- 完全試運行模式（--dry-run），可先預覽結果
- 自動建立備份，支援快速回滾
- 詳細日誌記錄（時間戳、統計數字、進度）
- 錯誤處理（檔案權限、路徑驗證、異常恢復）
- 獨立執行（不依賴 build 或其他工具）

## 使用方式

```bash
cd tools/kb-import

# 試運行 Phase 1
node phase1-dedup.js --source "H:/path/to/B" --dry-run

# 實際執行 Phase 1
node phase1-dedup.js --source "H:/path/to/B" --execute

# 同上 Phase 2
node phase2-dedup.js --source "H:/path/to/B" --dry-run
node phase2-dedup.js --source "H:/path/to/B" --execute
```

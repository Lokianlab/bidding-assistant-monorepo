## 概述

提供報價談判時的**底線計算**、**讓步區間**、**方案模擬**等工具，幫助標案團隊在協商中保護利潤空間、評估談判風險。

**核心場景**：
1. 案件方要求降價 → 快速計算底線 / 可讓步額度
2. 評估多個讓步方案 → 對比預期收益變化
3. 保護報價下限 → 確保不虧本

**依賴**：`src/lib/pricing/` 費用計算模組已完成

## 核心模型

### 報價結構

基於 pricing 模組的 `PricingSummary`（成本 + 管理費 + 稅），議價工具在此基礎上加入利潤和談判空間：

```
底線報價 = 成本 + 管理費 + 稅 + 最小利潤（e.g., 5%）
預案報價 = 底線報價 × (1 + 預期利潤率)    # 初始報價
目標報價 = 預案報價                       # 理想目標
可讓步額度 = 預案報價 - 底線報價
```

### 議價決策表

| 階段 | 報價 | 利潤率 | 狀態 | 說明 |
|------|------|--------|------|------|
| **底線** | `costBased` | `minMargin` (5%) | 危險 🔴 | 到此為止，再低就虧本 |
| **預案** | `proposedQuote` | `expectedMargin` (15%) | 可接受 🟢 | 初始報價 |
| **目標** | `targetQuote` | `idealMargin` (20%) | 最佳 🟢 | 理想目標 |
| **天花板** | `ceilingQuote` | `maxMargin` (30%) | 夢想 ✨ | 若對方願意 |

### 讓步方案模擬

用戶輸入「讓步額度」或「新報價」，系統計算對應的利潤率變化：

```
讓步 500k：
  原報價: 10,000,000
  新報價: 9,500,000
  新利潤率: 8.5%（預期 15% → 降 6.5 個百分點）
  風險評估: ⚠️ 低於預期，需評估是否接受
```

## API 設計

### 型別定義

```typescript
// src/lib/negotiation/types.ts

/** 成本基礎（來自 pricing 模組） */
export interface CostBase {
  directCost: number;      // 直接成本
  managementFee: number;   // 管理費
  tax: number;             // 營業稅
  subtotal: number;        // 小計（成本 + 管理費 + 稅）
}

/** 議價設定（應用設定的一部分） */
export interface NegotiationConfig {
  minMargin: number;       // 底線利潤率（e.g., 0.05 = 5%）
  expectedMargin: number;  // 預期利潤率（e.g., 0.15 = 15%）
  idealMargin: number;     // 理想利潤率（e.g., 0.20 = 20%）
  maxMargin: number;       // 天花板利潤率（e.g., 0.30 = 30%）
}

/** 報價方案 */
export interface QuoteScenario {
  name: string;            // 方案名稱（e.g., "底線", "預案", "讓步 500k"）
  quoteAmount: number;     // 報價金額
  profitAmount: number;    // 利潤金額
  profitRate: number;      // 利潤率（百分比，e.g., 0.15）
  status: 'safe' | 'warning' | 'danger' | 'dream';
}

/** 議價分析結果 */
export interface NegotiationAnalysis {
  costBased: QuoteScenario;           // 底線方案
  proposed: QuoteScenario;            // 預案
  target: QuoteScenario;              // 目標
  ceiling: QuoteScenario;             // 天花板
  scenarios: QuoteScenario[];         // 用戶模擬的其他方案
  allowanceAmount: number;            // 可讓步總額（預案 - 底線）
}
```

### 工具函式

```typescript
// src/lib/negotiation/helpers.ts

/** 計算底線報價（成本 + 最小利潤） */
export function calculateBaseline(
  costBase: CostBase,
  config: NegotiationConfig
): number;

/** 計算完整議價分析 */
export function analyzeNegotiation(
  costBase: CostBase,
  config: NegotiationConfig
): NegotiationAnalysis;

/** 模擬讓步方案 */
export function simulateCompromise(
  costBase: CostBase,
  config: NegotiationConfig,
  proposedAmount: number  // 新報價
): QuoteScenario;

/** 批量模擬多個讓步方案 */
export function simulateMultiple(
  costBase: CostBase,
  config: NegotiationConfig,
  concessions: number[]   // 讓步額度陣列
): QuoteScenario[];

/** 判斷報價安全性 */
export function getQuoteStatus(
  profitRate: number,
  config: NegotiationConfig
): 'safe' | 'warning' | 'danger' | 'dream';
```

### React Hook

```typescript
// src/lib/negotiation/useNegotiation.ts

/** 議價分析 Hook */
export function useNegotiation(
  costBase: CostBase | null,
  customConfig?: Partial<NegotiationConfig>
): {
  analysis: NegotiationAnalysis | null;
  isLoading: boolean;
  scenarios: QuoteScenario[];
  addScenario: (name: string, amount: number) => void;
  removeScenario: (index: number) => void;
  clearScenarios: () => void;
};
```

## UI 元件

### NegotiationPanel（主面板）

位置：`src/components/negotiation/NegotiationPanel.tsx`

功能：
- 顯示底線 / 預案 / 目標 / 天花板四個方案
- 快速切換不同方案視圖
- 讓步額度輸入 → 實時模擬
- 方案對比表

```tsx
<NegotiationPanel
  costBase={pricingSummary}
  onSelectScenario={(scenario) => handleApplyQuote(scenario)}
/>
```

### QuoteScenarioCard（方案卡片）

位置：`src/components/negotiation/QuoteScenarioCard.tsx`

顯示單一方案：
- 方案名稱 + 狀態徽章（安全 🟢 / 警告 ⚠️ / 危險 🔴）
- 報價金額、利潤率、利潤額
- 與預案的差異（△ -500k）

### ScenarioComparison（對比表）

位置：`src/components/negotiation/ScenarioComparison.tsx`

- 並排顯示多個方案
- 色彩編碼（安全/警告/危險）
- 可匯出 CSV

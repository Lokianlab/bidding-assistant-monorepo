# 06 開發進程

## 四層路線總覽

基於 00-總論的優先序和收入公式逆推，分四層推進：

| Layer | 目標 | 對應能力 | 狀態 |
|-------|------|---------|------|
| 0 | 工具就位，不需系統 | 知識庫精靈 + PCC MCP | 大部分完成 |
| 1 | 最直接影響得標率 | 能力 1（戰略）+ 能力 4（品質）+ 能力 2（知識庫升級） | 待開始 |
| 2 | 提升效率和體驗 | 能力 3（統一調度）+ 能力 6（排版+落標） | — |
| 3 | 錦上添花 | 能力 5（視覺生成） | — |

## Layer 0：工具就位

目標：讓現有工作流程立刻升級，不需要等系統完成。

| 項目 | 狀態 | 說明 |
|------|------|------|
| T1 知識庫精靈 | 已完成 | prompt v2.0（914 行）+ /kb Skill，CLI 對話建庫 |
| PCC MCP Server | 已完成 | 6 個工具，免費 API，PCC Monitor 每小時監控 |
| 00A-00E 初始內容 | 待開始 | 用 /kb 指令逐步建立五份知識庫 |

### PCC MCP 併入 Layer 0 的理由

1. **成本低**：API 免費、結構化、不需認證，實際開發 2 天
2. **價值高**：直接推動收入公式最前端（見下方分析）
3. **是戰略官的彈藥**：沒有 PCC 資料，Layer 1 的戰略官半盲

### Layer 0 剩餘工作

用 /kb 指令建立 00A-00E 初始內容（預估 2-3 天對話建庫）。完成後 Layer 0 收工，進入 Layer 1。

## Layer 1：最直接影響得標率

三個階段，按依賴順序推進。以 SaaS 網頁為主介面。

### P0：基礎建設

- SaaS 網頁骨架（認證 + 案件管理基本流程）
- 新增案件 → 案件列表 → 案件詳情
- Notion 資料庫整合（讀寫案件資料）
- 前提：確定 SaaS 產品架構

### P1：戰略分析 + 知識庫（能力 1 + 能力 2）

- L1 備標評估（戰略官 Agent）
- P 偵察報告自動生成
- 投標決策建議（投/不投，附可驗證依據）
- 知識庫搜尋 + 自動注入寫作脈絡
- PCC 情報自動整合（競爭對手、評委、價格）

### P2：品質閘門 + 撰寫（能力 4 + 部分能力 2）

- L2 備標規劃（企劃官 Agent）
- L3 章節撰寫
- 三道品質閘門（品管官 Agent：事實查核 / 需求對照 / 實務檢驗）
- 建議書匯出

## Layer 2-3（後續）

- **Layer 2**：統一 AI 調度中心（能力 3）+ 排版輸出 + 落標學習（能力 6）
- **Layer 3**：視覺生成整合（能力 5，目前用 Nanobanana 手動做）

## PCC API 對路線的戰略影響

PCC MCP 不只是技術工具，它直接影響收入公式每個環節。

### 收入公式衝擊

```
收入 = 得標案件數 × 平均案件金額 × 毛利率
```

| 公式變數 | PCC API 能做的事 | 影響程度 |
|---------|-----------------|---------|
| 發現機會數 ↑ | 自動監控新標案（listbydate + searchbytitle） | 中 |
| 投標決策通過率 ↑ | 競爭對手分析、機關偏好、預算行情、評委模式 | **高** |
| 得標率 ↑ | 評委背景→調整寫作策略、價格分析→報價策略 | **高** |
| 平均案件金額 ↑ | 篩選高價值案件 | 中 |

### 戰略官自動產出的情報（P1 備標評估）

| 分析項 | API 端點 | 決策影響 |
|--------|---------|---------|
| 同類案競爭激烈度 | searchbytitle 歷年同類案 | 判斷值不值得投 |
| 常見對手名單 + 得標率 | searchbycompanyname 交叉 | 知道要打贏誰 |
| 機關是否有默契廠商 | listbyunit + tender | 避免白做工 |
| 預算行情 / 底價比例 | tender 歷年金額 | 報價策略 |
| 評委可能人選 + 背景 | 同機關/同類案歷年評委 | **最高價值情報** |
| 評委-廠商相關性 | 三維矩陣分析 | 判斷勝算 |

### 對企劃撰寫的影響（P2）

知道評委背景後，企劃官可調整寫作策略：
- 學術背景→加強理論架構、文獻引用
- 實務背景→加強執行細節、過往成果
- 特定議題專家→摘要直接回應該議題

### 戰略官能力升級

```
原本：戰略官 = Notion 案件庫（自己的歷史）
現在：戰略官 = Notion 案件庫 + 全國 1,422 萬筆標案（整個市場）
```

從「看後照鏡開車」→「有 GPS 導航」。

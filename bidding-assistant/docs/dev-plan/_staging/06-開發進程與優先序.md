### PCC MCP 併入 Layer 0 的理由

1. 開發成本低（API 免費、結構化、不需認證，1-2 天）
2. 價值密度高（直接推動收入公式最前端）
3. 戰略官的基礎（沒有 PCC 資料，戰略官半盲）

```
原計畫：Layer 0（知識庫）→ Layer 1
建議：  Layer 0（知識庫 + PCC MCP）→ Layer 1（戰略官立刻有武器）
```

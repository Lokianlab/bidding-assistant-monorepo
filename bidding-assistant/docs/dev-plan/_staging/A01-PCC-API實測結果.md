## 內容

### API 基本資訊

- URL: `https://pcc-api.openfun.app/api/`
- 資料量：1,422 萬筆公告（1999/01 ~ 2026/02/13）
- 費用：免費，無需認證
- 限制：有 rate limit + 封鎖雲端 IP（GitHub Actions 403）

### 重大發現：API 有評委資料

僅限「公開評選/最有利標」的決標公告，包含：姓名、身份、序號、出席、經歷

### 分析能力矩陣

| 分析類型 | 可行性 | 備註 |
|---------|--------|------|
| 競爭對手得標率 | ✅ | 大員洛川 139 筆 |
| 機關-廠商偏好 | ✅ | listbyunit + tender |
| 價格競爭力 | ✅ | 含每家投標金額 |
| 類別市場分析 | ✅ | 食農教育 847 筆 |
| 評委交叉比對 | ✅ | 需逐筆爬 tender detail |
| 評委-廠商相關性 | ✅ | 需建三維矩陣 |
| 評委個別評分 | ❌ | 只有最終結果 |

### API 更新延遲

- 不是即時同步，是批次抓取
- 已建立監控系統（pcc-monitor/），用 Google Apps Script 每 4 小時檢查
- GitHub Actions 被 API 封鎖（403）

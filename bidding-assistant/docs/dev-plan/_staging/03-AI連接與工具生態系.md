### 決策脈絡

原本評估時 Gemini CLI 和 Codex CLI 都標「不裝」，後來用戶決定全部裝齊：

1. **Gemini CLI**：用戶是 Ultra 訂閱（~$22/月），免費 API quota 每日 2,000 次、每分鐘 120 次，作為備援 AI 和長上下文研究用，零額外成本
2. **Codex CLI**：用戶是 OpenAI Pro 訂閱（$200/月），作為備援 AI 和寫作/創意用，零額外成本
3. **NotebookLM MCP**：原暫存檔誤標「不裝」，實際 `.mcp.json` 早已配好（npx 自動拉），修正為「已配置」

決策原則：**既然都有現有訂閱，CLI 安裝零成本、零維護（npm install -g），沒理由不裝。** 多一個備援 AI 永遠比少一個好。

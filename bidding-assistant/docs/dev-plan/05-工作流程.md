# 05 工作流程

## 完整投標流程

```
招標文件
  → Step 0: PCC API 自動擷取（系統，1-2 分鐘）
  → Step 1: 系統組裝 P 偵察 prompt（系統，幾秒）
  → Step 2: Perplexity Deep Research（人貼，3-5 分鐘）
  → Step 3: P 偵察報告 + 招標文件 → L1 戰略分析（Claude）
  → Step 4: 團隊決策會議（投 or 不投）
  → L2 備標規劃
  → L3 企劃草稿
  → L4 定稿撰寫
  → L5 評選分析
  → L6 簡報規劃
  → L7 講稿腳本
  → L8 模擬演練
  → 投標
  → 結果回收 → 知識庫更新
```

## PCC API → P 偵察 → L1 整合（核心工作流）

### Step 0：使用者指定案件
輸入：案名 / 案號 / 機關名

### Step 1：PCC API 自動擷取（系統，1-2 分鐘）

```
searchbytitle(案名關鍵詞)
  → 同類型標案一覽
  → 每案用 tender() 取決標詳情
    → 得標者、未得標者、評委、金額

searchbycompanyname("大員洛川")
  → 我方投標歷史（139 筆）

searchbycompanyname(各競爭者名稱)
  → 對手投標歷史
```

產出：**預填數據包**（結構化表格）

### Step 2：組裝 P 偵察 prompt（系統，幾秒）

將預填數據包塞入 P 偵察報告模板：
- 「機關歷史標案一覽」已填 → Perplexity 只補政策脈絡
- 「競爭者名單」已填 → Perplexity 精準搜各家公司背景
- 「評委名單」已填 → Perplexity 搜各評委背景
- 「我方紀錄」已填 → Perplexity 只查相關新聞

產出：一份可直接貼的 prompt

### Step 3：Perplexity Deep Research（人，3-5 分鐘）

使用者複製 prompt → 貼入 Perplexity → 等待 → 匯出 .md

### Step 4：L1 戰略分析（Claude，10-15 分鐘）

輸入：招標文件 + P 偵察報告.md
使用 prompt：`01_戰略分析_v2.3.md`

L1 中依賴 P 偵察報告的步驟：
- Step 3 潛台詞分析 → 交叉比對政策脈絡
- Step 5 競爭格局 → 用真實得標者/投標者（非假想敵）
- Step 6 SWOT → 至少各一項引用 P 報告外部數據
- Step 7 投標決策 → 至少一個理由基於外部情報

### Step 5：團隊決策會議（人，30 分鐘）

基於 L1 報告做出：投 / 謹慎評估 / 放棄

## 使用者全程操作次數

| 步驟 | 操作 | 時間 |
|------|------|------|
| 指定案件 | 輸入關鍵詞 | 10 秒 |
| 貼 Perplexity | 複製 → 貼上 | 30 秒 |
| 等 Perplexity | 等待 | 3-5 分鐘 |
| 帶回 P 報告 | 複製 → 貼回 | 30 秒 |
| 餵招標文件 | 上傳/貼上 | 1 分鐘 |

**人只動手 3 次**：拿 prompt 出去、把結果帶回來、餵招標文件。

## AI 連接方式

| 工具 | 串接 | 角色 |
|------|------|------|
| Claude CLI + MCP | 主力 | 寫作/分析/品質檢查 |
| OpenAI CLI + MCP | 備援 | 寫作/創意 |
| Gemini CLI + MCP | 備援 | 研究/長上下文 |
| Perplexity MCP | 手動 | P 偵察報告（Deep Research） |
| Notion MCP | 已連 | 資料庫/知識管理 |
| Canva MCP | 可加 | 設計/排版/簡報 |

## L1-L8 階段定義

| 階段 | 名稱 | 觸發指令 | 階段 | 輸入 | 產出 |
|------|------|---------|------|------|------|
| L1 | 戰略分析 | /分析 | 投標 | 招標文件 + P 報告 | 戰略分析報告 + 提問清單 |
| L2 | 備標規劃 | /規劃 | 投標 | L1 報告 | 規劃書 |
| L3 | 企劃草稿 | /企劃 | 投標 | L2 規劃 | 企劃稿 + 預算表 |
| L4 | 定稿撰寫 | /撰寫 | 投標 | L3 企劃 | 各章初稿 |
| L5 | 評選分析 | /評選分析 | 評選 | L4 定稿 | 評選戰略 |
| L6 | 簡報規劃 | /簡報規劃 | 評選 | L5 分析 | 簡報架構 + 時間分配 |
| L7 | 講稿腳本 | /講稿腳本 | 評選 | L6 簡報 | 逐頁講稿 + Q&A |
| L8 | 模擬演練 | /模擬演練 | 評選 | L7 講稿 | 演練診斷報告 |

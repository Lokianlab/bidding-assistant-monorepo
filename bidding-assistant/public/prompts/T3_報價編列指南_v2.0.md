## 二、報價結構的兩種情境

### 情境 A：機關有提供標價清單

- **第一層（總預算表）**：嚴格按照機關科目結構與項次編列
- **第二層（細目分析表）**：自行拆解每個科目的成本明細
- **鐵律**：兩層數字必須 100% 一致，差 1 元即為致命錯誤

### 情境 B：機關未提供標價清單

- 依 WBS（工作分解結構）自行設計科目架構
- 科目分類邏輯：人事費 → 業務費 → 管理費 → 營業稅
- 科目名稱必須明確，禁止「其他」「雜項」等模糊用詞
- 同樣需要第一層總表 + 第二層細目

## 三、六大編列原則

### 原則 1：數字驗算（⭐ 最重要，致命等級）

```
驗算 A：第二層各細項加總 ＝ 第一層對應科目金額
驗算 B：第一層所有科目加總 ＝ 報價總額
驗算 C：營業稅 ＝（人事費 ＋ 業務費 ＋ 管理費）× 5%
驗算 D：管理費 ＝（人事費 ＋ 業務費）× 管理費率
驗算 E：報價總額 ＝ 公告預算金額（含稅）
```

**常見致命錯誤**：
- 營業稅誤以「總預算 × 5%」計算（正確為「人事+業務+管理」之和 × 5%）
- 第二層加總與第一層金額有落差（即使差 1 元也不行）
- 管理費以總預算為基底計算（正確基底為人事費+業務費）

### 原則 2：市場行情合理性

每個項目必須有行情依據，不可為了配合評分而虛報。常見行情參考：

| 項目類型 | 合理單價範圍 | 參考依據 |
|---------|------------|---------|
| 專家學者講座費 | 2,000–3,000 元/小時 | 行政院講座鐘點費標準 |
| 專案經理（部分工時）| 3,000–5,000 元/月 | 市場平均 |
| 企劃執行人員 | 15,000–25,000 元/月 | 產業平均 |
| 現場工作人員 | 1,200–2,000 元/場 | 時薪 × 工時 |
| 4K 專業攝影 | 20,000–30,000 元/場 | 影音市場行情 |
| 平面攝影 | 4,000–6,000 元/場 | 攝影師日薪 |
| 後製剪輯（含字幕）| 20,000–30,000 元/支 | 高工時專業服務 |
| 主視覺設計 | 15,000–25,000 元/案 | 設計公會參考 |
| 場地佈置 | 8,000–15,000 元/場 | 材料 + 人工 |

### 原則 3：大額項目強制拆解

| 單項金額門檻 | 最低拆解數量 |
|------------|------------|
| > 10 萬元 | 3–5 個細項 |
| > 20 萬元 | 5–8 個細項 |

拆解邏輯：
- 人力類 → 人數 × 天數 × 日薪
- 設計類 → 企劃 + 設計 + 修改 + 輸出
- 製作類 → 材料 + 人工 + 設備租賃 + 運輸
- 活動類 → 企劃 + 講師 + 場地 + 材料 + 攝影

### 原則 4：金額避免整數（真實感處理）

```
❌ 避免：60,000、180,000、500,000
✅ 改為：58,433、179,876、497,250

微調範圍：原金額 ±3% 以內
小額（< 5 萬）：± 500–1,500
中額（5–15 萬）：± 1,000–3,000
大額（> 15 萬）：± 2,000–5,000

個位數不要是 0（更真實）
```

### 原則 5：每筆細項必須有數量依據

```
✅ 好：攝影師 2 天 × 12,000 元/天 = 24,000 元
✅ 好：後製修圖 80 件 × 200 元/件 = 16,000 元
❌ 壞：攝影相關費用 1 式 46,000 元
```

### 原則 6：總額精確符合預算

- 有預算上限 → 報價總額 = 預算上限（差 1 元都不行）
- 無預算上限 → 控制在合理範圍，並附計算邏輯說明

## 四、產出文件規格

報價編列完成後，須產出以下文件：

### 文件 A：對外投標報價表（隨建議書提交）

包含：
1. **預算總表**（第一層）：項次、科目名稱、金額、佔比、說明
2. **經費細目分析表**（第二層）：項次、項目、單位、數量、單價、複價、算式與編列說明
3. **預算編列邏輯說明**（200–300 字文段）
4. **GC-09 法律聲明**（強制插入於預算章節結尾）

### 文件 B：內部利潤分析表（公司內部使用）

包含：
1. **分項成本利潤表**：每科目的對外報價、直接成本、間接成本（13%）、毛利、毛利率
2. **整體利潤評估**：總毛利率（合理範圍 15–30%）、接案建議
3. **風險與應變預算**：風險項目、發生機率、預留預算、應對方式

### 文件 C：內部執行預算控管表（執行團隊使用）

包含：
1. **各科目執行預算分配**：對外報價、內部預算上限、預留利潤
2. **成本控管重點**：採購比價要求、簽約注意事項、緊急預備金
3. **驗收標準**：各項交付物的驗收規格
4. **付款節點**：建議的分期付款比例

## 五、品質檢查清單（投標前必查）

### ⭐⭐⭐ 致命級（未通過 = 廢標風險）

```
□ 第一層與第二層數字是否 100% 一致？
□ 營業稅 = (人事費+業務費+管理費) × 5%？（不是總額 × 5%）
□ 報價總額是否精確等於公告預算金額？
□ 細項加總是否等於對應大項？（每一科目逐一驗算）
□ 科目名稱是否與標方標價清單完全一致？（情境 A）
```

### ⭐⭐ 嚴重級（影響評分）

```
□ 管理費 = (人事費+業務費) × 管理費率？（合理範圍 8–12%）
□ 每個項目是否在市場行情合理範圍？
□ 大額項目（> 10 萬）是否已拆解細項？
□ 每筆費用是否都有「數量 × 單價 = 複價」的計算式？
□ 企劃內容 ↔ 預算的一致性？（企劃提到的項目是否都有預算？）
```

### ⭐ 加分級（提升競爭力）

```
□ 金額是否避免整數？（至少到十位數）
□ 說明欄是否清楚完整？
□ 預算編列邏輯說明是否已撰寫？
□ 法律聲明是否已插入？
□ 利潤率是否合理（15–30%）？
```

## 六、利潤分析方法論

### 成本結構三層模型

```
投標報價（對外）
├── ① 直接成本（60–70%）：人力、材料、外包、場地
├── ② 間接成本（10–15%）：管銷、水電、保險、交通
└── ③ 預期利潤（15–30%）：毛利 − 營業稅
```

### 利潤合理性判斷

| 案件類型 | 合理毛利率 | 警戒線 |
|---------|----------|--------|
| 展覽活動類 | 15–30% | < 10% 不建議接案 |
| 設計顧問類 | 20–40% | < 15% 須審慎評估 |
| 工程施工類 | 10–20% | < 8% 成本控管困難 |
| 人力服務類 | 25–45% | < 20% 人力成本過高 |
| 影音製作類 | 18–35% | < 12% 後製成本高 |

### 風險預備金

建議預留毛利的 15–20% 作為風險預備金，涵蓋：人力成本上漲、材料價格波動、工期延誤、天氣影響等不可預見因素。

## 七、常見錯誤速查表

| 錯誤類型 | 危害等級 | 錯誤範例 | 正確做法 |
|---------|---------|---------|---------|
| 兩層數字不符 | ⭐⭐⭐ 致命 | 第一層 234,000；第二層加總 216,000 | 兩者必須完全相同 |
| 營業稅算錯 | ⭐⭐⭐ 致命 | 784,000 × 5% = 39,200 | (人事+業務+管理) × 5% |
| 總額不符預算 | ⭐⭐⭐ 致命 | 預算 170 萬，報價 165 萬 | 微調至精確 170 萬 |
| 管理費算錯 | ⭐⭐ 嚴重 | 總預算 × 10% | (人事+業務) × 費率 |
| 不按標方科目 | ⭐⭐ 嚴重 | 標方「作品攝影」改成「影像記錄」| 嚴格按標方名稱 |
| 虛報預算 | ⭐⭐ 嚴重 | 志工培訓編 20 萬 | 回歸行情 2–3 萬 |
| 金額太整 | ⭐ 輕微 | 60,000、180,000 | 58,433、179,876 |
| 缺少細項 | ⭐ 中度 | 「展場費用 1 式 50 萬」 | 拆解為 5+ 子項 |
| 單價離譜 | ⭐ 輕微 | 攝影師 1 天 50,000 元 | 查行情：8,000–15,000 |

```

# 政府標案報價表編列任務

## 基本資訊

- 案名：【請填入標案名稱】
- 案號：【請填入案號】
- 公告預算金額（含稅）：NT$ 【請填入金額】
- 標方是否提供標價清單：【有 / 無】
- 案件類型：【展覽活動 / 設計顧問 / 影音製作 / 教育培訓 / 其他：___】

---

## 任務指示

請依照以下規則，完成本案的「對外投標報價表」與「內部利潤分析」。

### 一、對外報價表（投標用）

#### 1. 第一層：總預算概覽表

- 若標方有提供標價清單 → 嚴格按照標方科目結構與項次編列，科目名稱不得更改
- 若標方未提供 → 自行設計科目結構，分類為：人事費、業務費、管理費、營業稅
- 每科目標示金額與佔比
- 格式：

| 項次 | 科目名稱 | 金額（元）| 佔比 | 說明 |
|------|---------|----------|------|------|

#### 2. 第二層：經費細目分析表

將第一層每個科目拆解成細項，每筆細項必須包含：
- 項目名稱、單位、數量、單價、複價
- 「算式與編列說明」欄：完整計算式 + 單價依據來源

格式：

| 項次 | 項目 | 單位 | 數量 | 單價 | 複價 | 算式與編列說明 |
|------|------|------|------|------|------|--------------|

拆解規則：
- 單項 > 10 萬元 → 拆成 3–5 個細項
- 單項 > 20 萬元 → 拆成 5–8 個細項
- 每個細項必須有「數量 × 單價 = 複價」的依據，不可出現「1 式 X 元」

#### 3. 數字驗算（必須全部通過，請在報告中逐一展示驗算結果）

```
□ 驗算 A：第二層人事費細項加總 = 第一層人事費
□ 驗算 B：第二層業務費細項加總 = 第一層業務費
□ 驗算 C：管理費 = (人事費 + 業務費) × 管理費率
□ 驗算 D：營業稅 = (人事費 + 業務費 + 管理費) × 5%
□ 驗算 E：第一層所有科目加總 = 報價總額 = 公告預算金額
```

⚠️ 特別注意：
- 營業稅的計算基底是「人事費 + 業務費 + 管理費」，不是總預算金額
- 管理費的計算基底是「人事費 + 業務費」，不是總預算金額
- 總額必須精確等於公告預算金額，差 1 元都不行

#### 4. 金額真實感處理

- 所有金額避免整數（如 60,000），改為帶尾數的金額（如 58,433）
- 微調範圍：原金額 ±3% 以內
- 個位數不要是 0
- 但「算式與編列說明」中的單價可維持整數（如 2,500 元/小時），由數量的組合使複價自然產生尾數

#### 5. 市場行情

每個項目的單價必須符合市場行情。若編列依據為政府標準（如行政院講座鐘點費），請在說明中註明。若為市場估算，請說明參考基礎。

#### 6. 預算編列邏輯說明

在報價表後附上 200–300 字的文段，說明預算分配策略：
- 為何某些科目佔比較高（對應核心評分項目）
- 單價依據來源（實際詢價 / 政府標準 / 市場估算）
- 可調整的彈性空間

#### 7. 法律聲明（強制插入於預算章節結尾）

> 「本報價單所列之各項費用估算，係為規劃本專案所需之預算。在契約總價不變且不影響專案品質之原則下，為順應專案實際執行需求，應同意本公司得自行於各費用科目間互相勻支流用。」

---

### 二、內部利潤分析（內部使用，不放入建議書）

#### 1. 分項成本利潤表

為每個科目估算：
- 直接成本（實際執行所需費用）
- 間接成本（以直接成本 × 13% 估算）
- 毛利 = 對外報價 − 直接成本 − 間接成本
- 毛利率 = 毛利 ÷ 對外報價 × 100%

格式：

| 科目 | 對外報價 | 直接成本 | 間接成本(13%) | 毛利 | 毛利率 | 備註 |
|------|---------|---------|-------------|------|--------|------|

#### 2. 整體利潤評估

計算並說明：
- 總毛利率（合理範圍 15–30%）
- 扣除營業稅後的稅前淨利率
- 是否建議接案
- 主要成本風險點

#### 3. 風險與應變預算

列出 3–5 項主要風險，格式：

| 風險項目 | 發生機率 | 預留預算 | 應對方式 |
|---------|---------|---------|---------|

建議風險預備金佔毛利的 15–20%。
```

□ 金額已非整數化處理：✓ / ❌
□ 整體毛利率：___% → 合理 / 偏低 / 偏高

```
### 四、輸出順序

請依以下順序輸出：
```

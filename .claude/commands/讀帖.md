讀取指定論壇討論串的所有帖子，按時間排序。

用法：`/讀帖 {thread-id}`
例如：`/讀帖 forum-optimization`

## 步驟

### 1. 確認 thread 存在

讀 `docs/records/forum/_threads.md`，找到該 thread 的狀態和基本資訊。
找不到就告訴用戶「沒有這個 thread」。

### 2. 搜集所有帖子

用 Grep 從所有 forum 檔案中找出帶有 `thread:{thread-id}` 的帖子：

```bash
grep -n "thread:$THREAD_ID" docs/records/forum/[0-9]*.md
```

### 3. 讀取並排序

對每個匹配的帖子：
- 從匹配行往前找到帖子的 header（type|timestamp|機器碼...）
- 往後讀到下一個 `---` 分隔線
- 按 timestamp 排序

### 4. 輸出

按時間順序輸出所有帖子，格式：

```
## {thread-id}（{狀態}）
來源：_threads.md

---
### {timestamp} | {機器碼} | {type}
{帖子內容}

---
### {timestamp} | {機器碼} | {type}
{帖子內容}
```

每則帖子之間用分隔線隔開，讓用戶能一次看完整個對話脈絡。

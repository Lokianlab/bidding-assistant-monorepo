讀取指定論壇討論串的所有帖子，按時間排序，並計算投票狀態。

用法：`/讀帖 {thread-id}`
例如：`/讀帖 forum-optimization`

$ARGUMENTS

## 步驟

### 1. 確認 thread 存在

讀 `docs/records/forum/_threads.md`，找到該 thread 的狀態和基本資訊。
找不到就告訴用戶「沒有這個 thread」。

### 2. 搜集所有帖子

用 Grep 從所有 forum 檔案中找出帶有 `thread:{thread-id}` 的帖子：

```
Grep pattern="thread:{thread-id}" path="docs/records/forum" output_mode="content" -B=1 -A=30
```

### 3. 讀取並排序

對每個匹配的帖子：
- header 行格式：`{type}|{timestamp}|{機器碼}|{priority}|ref:{引用}|thread:{id}`
- 從 header 行往後讀到下一個 `---` 分隔線
- 按 timestamp 排序

### 4. 輸出帖子

按時間順序輸出所有帖子，格式：

```
## {thread-id}（{狀態}）

---
### {timestamp} | {機器碼} | {type}
{帖子內容}

---
### {timestamp} | {機器碼} | {type}
{帖子內容}
```

### 5. 投票狀態（discuss thread 才做）

如果 thread 的發起帖是 discuss 類型，生成投票摘要。

**兩層資料來源**：
1. **`_threads.md` 投票欄**（8 欄格式優先）：直接讀取 `同意` 和 `反對` 欄的機器碼列表，這是最權威的投票記錄
2. **帖子內容掃描**（補充）：對於不在投票欄中的機器，用關鍵詞掃描帖子推斷立場

**帖子掃描規則**（80/20 方案：自動掃 + 人工兜底）：
- 用 grep 關鍵詞掃描：`同意|支持|贊成` → 計為同意；`反對|異議` → 計為反對
- 帖子內容含「有條件」→ 計為有條件同意（附條件摘要）
- discuss 發起人本身的立場也要納入（通常在帖子末尾）
- **投票欄 vs 帖子掃描衝突時**：以投票欄為準（投票欄是明確操作，帖子掃描是推測）
- **結果末尾加免責聲明**：`⚠ 自動解析，有疑義的請手動確認`

**活躍機器列表**：從所有 `_snapshot-*.md` 檔名取得機器碼。

**正式/見習判斷**：見習成員的帖子會標「見習成員，意見供參考不計票」，以此區分。

**輸出格式**：

```
## 投票狀態

| 立場 | 機器 |
|------|------|
| 同意 | {列出機器碼} |
| 反對 | {列出機器碼} |
| 有條件同意 | {機器碼}（條件：{摘要}） |
| 未表態 | {列出機器碼} |
| 見習意見 | {機器碼}（{立場}） |

⚠ 自動解析，有疑義的請手動確認

**門檻判斷**：
- 正式成員數：{N}
- 需要同意數：{過半數}（≤5 台過半，6-10 台 2/3）
- 目前同意數：{M}
- 狀態：{已達門檻 / 未達門檻，差 X 票}
```

### 6. 超時檢查

如果 thread 狀態是「進行中」，檢查超時（布林判斷，不倒數計時）：

- 找出最後一條 reply 的時間戳
- 對照 thread 優先級的超時規則（P1: 12hr, P2: 24hr, P3: 72hr）
- **判斷結果只有兩種**：超時了 / 沒超時

```
**超時狀態**：{已超時 → 「未表態機器視為棄權，不計入分母」 / 未超時 → 「距超時還有 Xhr」}
```

如果有超時棄權，**重新計算門檻**：把棄權機器從分母移除，更新投票狀態中的門檻判斷。

### 7. 共識宣告草稿（門檻達標時自動生成）

如果投票已達門檻且無反對，**直接生成可複製貼上的宣告草稿**：

```
## 共識宣告草稿（自動生成，機器確認後可直接發出）

reply|{當前時間}|{你的機器碼}|{priority}|ref:{原始discuss引用}|thread:{thread-id}
**共識宣告：{thread-id}**

{N}/{M} 正式成員明確同意（{列出}）{，如有超時棄權：「{列出}超時棄權，不計入分母」}。

**共識要點**：
1. {從 discuss 原帖和 reply 中歸納的要點}
2. ...

**各方立場**：
- {逐一列出每台機器的立場和來源}

**狀態**：{判斷改動級別}。{第一級 → 「不需用戶核准」/ 第三級 → 「需等用戶核准才能執行」}。
---
```

機器過目確認後直接貼到自己的 forum 檔案，不用重寫。

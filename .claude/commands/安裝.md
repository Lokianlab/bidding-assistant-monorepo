在全新機器上安裝完整開發環境。逐步引導使用者完成設定。

## 步驟

### 1. 檢查基礎工具

逐一檢查以下工具是否已安裝，未裝的幫使用者安裝：

```bash
node --version    # 需要 v22+
git --version     # 需要 Git
gh --version      # 需要 GitHub CLI
claude --version  # 需要 Claude Code CLI
gemini --version  # 需要 Gemini CLI
codex --version   # 需要 Codex CLI
```

未安裝的執行：
- `winget install OpenJS.NodeJS.LTS`
- `winget install Git.Git`
- `winget install GitHub.cli`
- `npm install -g @anthropic-ai/claude-code`
- `npm install -g @google/gemini-cli`
- `npm install -g @openai/codex`

如有安裝新工具，提醒使用者**關閉並重開終端機**再繼續。

### 2. 登入 GitHub

```bash
gh auth login
# 引導選：GitHub.com → HTTPS → Yes → Login with a web browser
```

### 3. 下載專案

```bash
mkdir -p /c/dev
gh repo clone Lokianlab/bidding-assistant-monorepo /c/dev/cc程式
cd /c/dev/cc程式
```

### 4. Git 設定

```bash
git config user.name "Jin"
git config user.email "gasklath20312@gmail.com"
```

### 5. 安裝依賴

```bash
cd /c/dev/cc程式/bidding-assistant && npm install
```

### 6. 環境變數

```bash
cp bidding-assistant/.env.example bidding-assistant/.env.local
```
提醒使用者手動編輯 `.env.local`，填入 `NOTION_API_KEY` 和 `NOTION_DATABASE_ID`。

### 7. 驗證

```bash
cd /c/dev/cc程式/bidding-assistant
npm test          # 預期：26 files, 560 tests 全過
npm run build     # 預期：0 errors
```

### 8. 完成

告知使用者：以後開工只要 `cd C:\dev\cc程式` 然後 `claude` 就好。
